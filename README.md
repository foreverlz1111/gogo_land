# 安装
从[[官网](https://golang.org )]下载对应系统的tar.gz包

解压包内文件到usr/local/go然后添加用户变量：

```
echo export PATH=$PATH:/usr/local/go/bin >> ~/.bashrc
```

重启电脑，然后尝试一下：

```
go version
```

运行代码

```
go run main.go
```

**翻译自[Go 网站](https://tour.go-zh.org)**
